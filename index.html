Saving your game

Faxanadu has no battery save. Progress is kept two ways:

  • Mantras — the in-game password system. Visit a Guru and write down the mantra he gives you; enter it at the title screen to continue. This is the authentic 1989 experience.
  • Save states — instant snapshots stored in this browser. Pick a slot, hit F5 to save and F8 to restore. States survive closing the tab, but live only in this browser profile.

About this project

Faxanadu (Hudson Soft / Falcom, 1987) is still under copyright, so this player is distributed without the game itself. You supply a ROM dumped from your own cartridge; the emulator (jsnes, MIT license) runs it entirely client-side. No servers, no tracking, no network use at all after the page loads — save the page and it keeps working offline, indefinitely.
